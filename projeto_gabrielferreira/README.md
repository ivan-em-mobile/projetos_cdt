# 💈 Barbearia Andrades - Sistema de Atendimento via Terminal

Este é um sistema interativo em Python que roda diretamente no terminal, projetado para gerenciar as operações básicas de atendimento da *Barbearia Andrades*. Ele permite que os clientes agendem horários, cancelem agendamentos, visualizem a lista de serviços e preços, acessem a localização/contato e avaliem a experiência.

---

## 🚀 Funcionalidades

O sistema conta com um menu interativo contendo as seguintes opções:

1. *Agendar horário*: Registra o nome do cliente, a data e o horário desejado para o atendimento.
2. *Cancelar agendamento*: Permite a exclusão do agendamento ativo mediante confirmação de segurança do nome do cliente.
3. *Serviços disponíveis*: Exibe o catálogo atualizado de cortes, barbas, tratamentos e seus respectivos valores.
4. *Localização e contato*: Mostra o endereço físico do estabelecimento e o telefone de suporte/contato.
5. *Avalie nosso serviço*: Coleta o feedback do cliente com base em uma escala de 1 a 5, retornando mensagens personalizadas.
0. *Sair*: Encerra a execução do programa com segurança.

---

## 📋 Pré-requisitos

Para executar este projeto, você só precisará do Python instalado em sua máquina. 

* *Python 3.x* ou superior.

Não há necessidade de instalar nenhuma biblioteca externa (o projeto utiliza apenas o módulo nativo os).
