## Sobre o Projeto

Este projeto foi desenvolvido em equipe, com o objetivo de criar um sistema para uma sorveteria. Ele possui funções para pesquisar opções de sabores, mudar entre temas de dia e noite, verificar catálogo, cadastrar usuário e seção de ajuda. 

## Como Foi Desenvolver o Projeto

Realizar esse projeto em equipe foi uma experiência boa. Cada integrante ajudou de alguma forma, seja na programação, na organização ou na resolução dos problemas que apareceram durante o desenvolvimento.

## Dificuldades Encontradas

Uma das maiores dificuldades do projeto foi organizar todas as partes do sistema e fazer com que tudo funcionasse corretamente. Em alguns momentos tivemos dificuldades com erros no código e também na hora de conectar algumas funcionalidades.

Outra dificuldade foi dividir o tempo para conseguir terminar todas as tarefas do projeto dentro do prazo. Porém, com paciência e trabalho em equipe, conseguimos resolver os problemas.

## O Que Foi Mais Simples

A parte mais simples foi desenvolver as ideias principais do projeto, porque todos do grupo participaram dando sugestões. Também foi mais fácil trabalhar nas partes visuais e na organização das informações da sorveteria.

## Conclusão

Esse projeto foi muito importante para nosso aprendizado, porque conseguimos colocar em prática vários conhecimentos de programação. Trabalhar em equipe ajudou bastante no desenvolvimento do projeto e tornou a experiência mais interessante e produtiva.

Através desse projeto aprendemos não apenas sobre programação, mas também sobre colaboração, responsabilidade e resolução de problemas.
